# What makes a knowledge system effective?

A system for organizing knowledge must satisfy multiple criteria simultaneously: it should preserve information fidelity, enable rapid retrieval, support connection-making, and remain maintainable over time.

## Considerations
- Fidelity (does the system preserve nuance?)
- Speed (can I find what I need quickly?)
- Serendipity (does the system surface unexpected connections?)
- Maintenance (can one person sustain it?)
